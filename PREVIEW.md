## 🛠️ 技术架构

### 前端 (Next.js 15)
- **框架**: Next.js App Router
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **状态管理**: React Hooks

### 后端 (API Routes)
- **数据库**: SQLite (better-sqlite3)
- **API设计**: RESTful
- **认证**: 可扩展的JWT认证
- **文件存储**: 本地文件系统

### 数据库设计
```sql
-- 笔记表
CREATE TABLE notes (
  id INTEGER PRIMARY KEY,
  title TEXT NOT NULL,
  content TEXT NOT NULL,
  category TEXT,
  tags TEXT,
  created_at DATETIME,
  updated_at DATETIME
);

-- 对话表  
CREATE TABLE conversations (
  id INTEGER PRIMARY KEY,
  session_id TEXT NOT NULL,
  role TEXT NOT NULL,
  content TEXT NOT NULL,
  metadata TEXT,
  created_at DATETIME
);

-- 记忆表
CREATE TABLE memories (
  id INTEGER PRIMARY KEY,
  key TEXT UNIQUE NOT NULL,
  value TEXT NOT NULL,
  category TEXT,
  importance INTEGER,
  created_at DATETIME,
  updated_at DATETIME
);
```

## 🚀 快速启动

### 本地运行
```bash
# 1. 进入项目目录
cd /tmp/second-brain

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问 http://localhost:3000
```
