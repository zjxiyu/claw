# 🧠 第二大脑系统

一个智能的个人知识管理系统，用于查看和管理所有的笔记、对话和记忆。

## ✨ 功能特性

### 📝 笔记管理
- 创建、编辑、删除笔记
- 分类和标签管理
- 富文本内容支持
- 实时搜索

### 💬 对话记录
- 记录与AI的对话历史
- 按会话分组
- 对话内容分析
- 重要对话标记

### 🧠 记忆库
- 关键信息存储
- 重要性分级（1-5星）
- 分类管理
- 快速检索

### 🔍 智能搜索
- 全文搜索
- 跨模块搜索
- 实时结果
- 搜索历史

## 🚀 快速开始

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

### 部署到Vercel
1. 将代码推送到GitHub
2. 在Vercel导入项目
3. 配置域名：claw.20803960.xyz
4. 点击部署

## 📊 技术架构

- **前端**: Next.js 15 + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: SQLite (better-sqlite3)
- **部署**: Vercel
- **域名**: claw.20803960.xyz

## 🎯 使用场景

1. **知识管理**: 整理学习笔记、工作文档
2. **对话存档**: 保存重要的AI对话记录
3. **记忆存储**: 记录关键信息、密码提示等
4. **项目规划**: 管理项目思路和进度
