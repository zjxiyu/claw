### 步骤1: 创建GitHub仓库
1. 访问 https://github.com/new
2. 创建新仓库: `second-brain`
3. 描述: "个人知识管理系统"
4. 选择: Public (公开)
5. 点击 "Create repository"

### 步骤2: 推送代码到GitHub
```bash
# 在项目目录执行
cd /tmp/second-brain

# 添加远程仓库（替换YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/second-brain.git

# 重命名分支并推送
git branch -M main
git push -u origin main
```

### 步骤3: Vercel部署
1. 访问 https://vercel.com
2. 点击 "Add New..." → "Project"
3. 导入你的GitHub仓库 `second-brain`
4. 点击 "Import"

### 步骤4: 配置域名
1. 在Vercel项目设置中，点击 "Domains"
2. 添加域名: `claw.20803960.xyz`
3. 按照提示配置DNS记录
4. 等待SSL证书自动签发

## 🚨 故障排除

### 常见问题
1. **域名无法访问**
   - 检查DNS配置是否正确
   - 等待DNS传播（最多24小时）
   - 检查Vercel项目状态

2. **页面加载错误**
   - 清除浏览器缓存
   - 检查控制台错误信息
   - 确认Vercel部署成功

3. **功能不工作**
   - 检查JavaScript是否启用
   - 确认网络连接正常
   - 查看浏览器控制台
